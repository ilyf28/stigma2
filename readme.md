## STIGMA

STIGMA is monitoring solution for cloud.
STIGMA use a nagios , grafana , influxDB for monitoring resource.

### INSTALLATION 

* composer install 
* make .env in ROOT

### REQUIRE
* php > 5.5
* mysql > 5

### Testing 
Stigma project use codecept to run tests. 

For running test,

* Make a file  .env.testing (ref to .env.example) 
* bin/codecept run
